# Get Started — Joule

> SAP Fiori for iOS | Joule

## What is Joule?

Joule is SAP's generative AI copilot — the new interface for the AI era. It provides mobile users with contextual insights across SAP solutions, integrating third-party data as needed, delivered in natural language. As Joule evolves, it serves as the central entry point for multi-agent systems.

SAP's AI strategy delivers **relevant, reliable, and responsible AI** through two complementary approaches:

| Approach | Description |
|----------|-------------|
| **In-App AI** | Contextual, embedded components and patterns powered by SAP AI Foundation — in-context within existing workflows |
| **Joule** | Conversational copilot for open-ended, cross-module tasks via natural language |

---

## In-App AI vs. Joule

**In-App AI** — real-time recommendations and proactive support within the natural workflow. Fast, focused, contextual. Enables faster processes, improved productivity, enhanced decision-making.

**Joule** — delivers AI-driven business insights tailored to user needs. Handles complex, multi-step, cross-module tasks. Users stay in control — reviewing and validating AI-generated information before acting.

→ See [ai-handoff.md](ai-handoff.md) for guidance on transitioning between the two.
