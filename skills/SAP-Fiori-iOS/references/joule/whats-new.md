# What's New — Joule

> SAP Fiori for iOS | Joule

This article tracks additions and changes to the Joule client for iOS, organized by release.

---

## Joule client, iOS 26.4

### Components

**Input Field** — Feature Enhancement
Now includes mode selection, assistant, dictation, and voice controls. Pills display the user's selected grounding context, giving clearer visibility into what context is shaping the conversation.

**Panel** — Feature Enhancement
Refreshed with a cleaner visual header. More refined and consistent look aligned with the latest Joule design language.

**Response Actions** — Feature Enhancement
Users can now copy or regenerate any response directly from the message, making it easier to reuse output or request a fresh answer without retyping the prompt.

### Patterns

**Conversations List** — Feature Enhancement
Users can now search and sort their conversations, making it easier to find past threads and stay organized across multiple ongoing interactions.

**Dictation** — New Pattern
Users can dictate messages using the device's built-in dictation capability, enabling hands-free input for faster, more accessible conversational workflows.

**Typeahead** — New Pattern
As users begin typing, Joule surfaces contextual suggestions to complete prompts faster, reducing keystrokes and supporting more fluid interactions.

**Welcome Screen** — Feature Enhancement
Updated with a refreshed logo and cleaner visual design for a more welcoming, on-brand first impression.

---

## Joule client, iOS 26.1

### Components

**Likert Scale** — New Component
Provides a conversational-friendly response type for expressing nuanced feedback with minimal effort, while capturing consistent structured data for enterprise insights.

### Patterns

**Initial Loading** — Feature Enhancement
Users can now begin typing their prompt during the loading phase, reducing perceived wait time and supporting faster, more fluid conversational workflows.

**Multi-Threading** — New Pattern
Threads allow users to have multiple ongoing conversations with Joule — each focused on a different task or topic — enabling easy context switching and returning to prior threads.

**Persistent Attachment** — New Pattern
Users can attach files to their Joule conversations by tapping the file upload button next to the input field.

**Transparency and Explainability** — Feature Enhancement
Sources are now displayed separately from response insights, providing clearer and more direct access to original information.

## Joule client, iOS 25.11

### Components

**Response Actions** — Guideline Enhancement
Article updated with accessibility guidance.

### Patterns

**Confirmation** — New Pattern
Used to verify Joule correctly understood a user's request before taking action, especially for transactions or commands that are difficult to undo.

**Loading** — New Pattern
Previously "Loaders". Now includes new streaming patterns: text streaming, skeleton loading, and a "scroll for more" button.
