# SAP Fiori for watchOS — UI Kit

> Resources

## What is it?

The SAP Fiori for watchOS Figma UI Kit contains UI components, patterns, page types, and foundational elements (colors, typography) for watchOS. Use it to accelerate design and ensure consistency across SAP watch applications.
