# SAP Fiori Mentor App — Resources

> Resources

## What is it?

The SAP Fiori Mentor app is an interactive playground for mobile designers and developers. It enables live previews of all SAP BTP SDK UI components with adjustable parameters, and lets you copy ready-made code snippets directly from the app.

Available for **iPad** (App Store) and **Android** (Google Play).

---

## Download

- **iOS (iPad):** [App Store](https://apps.apple.com/de/app/sap-fiori-mentor/id1215284965)
- **Android:** [Google Play](https://play.google.com/store/apps/details?id=com.sap.mobile.mentor.android)

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Live component preview** | Adjust settings and see behavior/interactions in real time |
| **Code export** | Export code samples with predefined settings for any UI component |
| **Phone & tablet simulation** | Preview components in both form factors |
| **Backend components** | Onboarding flows, SAP Foundation & OData exploration |
| **Best practices** | Usage guidance per component |

---

## Who It's For

**Designers:** Understand the full scope of component functionality, specify designs with applicable parameters, hand off code snippets to developers.

**Developers:** Export ready-made code with customized settings directly into your development tool.
