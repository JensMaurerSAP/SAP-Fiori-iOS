# SAP Fiori for iOS — UI Kit

> Resources

## What is it?

The SAP Fiori for iOS Figma UI Kit contains UI components, patterns, page types, and foundations for SAP Fiori iOS. It accelerates design and development and encourages consistency across SAP applications.
