# Joule UI Kit, iOS — Resources

> Resources

## What is it?

The Joule UI Kit for iOS contains UI components, patterns, and foundations for Joule — SAP's generative AI-based copilot. Use it alongside the main SAP Fiori for iOS UI Kit when designing Joule experiences.
