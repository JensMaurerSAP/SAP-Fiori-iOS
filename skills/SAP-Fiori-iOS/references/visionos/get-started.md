# Get Started — SAP Fiori for visionOS

> SAP Fiori for visionOS | Get Started
> Based on visionOS 2.0

## What is it?

SAP Fiori for visionOS adapts SAP software for spatial computing on Apple Vision Pro. Designing for visionOS requires understanding spatial design principles before applying SAP Fiori-specific guidance.
