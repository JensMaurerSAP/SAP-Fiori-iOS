# Mobile UX Consistency — Discover

> SAP Fiori for iOS | Discover

## What is it?

Consistent UX across SAP mobile applications reduces training, support, development, and maintenance costs. SAP's mobile design system follows platform-specific characteristics (iOS/iPadOS for Apple platforms) rather than a single cross-platform Web Product Standard.

> Mobile applications don't follow a specific UXC Product Standard, but SAP Fiori for iOS guidelines are crafted to achieve consistent mobile UX, especially when transitioning from web-based UXC principles.
