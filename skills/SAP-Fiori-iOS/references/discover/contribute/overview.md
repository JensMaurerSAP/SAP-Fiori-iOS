# Contribute — Overview

> SAP Fiori for iOS | Discover / Contribute

## What is InnerSource for Mobile?

InnerSource for Mobile is a platform for reusable community components and extensions for the SAP Design System and SAP BTP SDK for iOS and Android. It enables teams across SAP to build on each other's work and contribute back to the shared mobile design system.

**GitHub:** InnerSource for Mobile on GitHub

---

## Two Component Types

| Type | Owner | Repository |
|------|-------|-----------|
| **Common Components** | Core Design & SDK Team | Main SAP BTP SDK |
| **Community Components** | Contributing team | Community Components extension |

Contributing to community components does **not** automatically guarantee inclusion in the common Mobile Design System — that decision rests with the Core Product Teams.

---

## Contribution Types

| Type | Description |
|------|-------------|
| **New Component** | Add a new community component |
| **Component Extension** | Extend an existing common component |

Contribute UI design **together with code** whenever possible. Cross-platform parity (iOS + Android) is encouraged.
